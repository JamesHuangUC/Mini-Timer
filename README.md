# Mini Timer

Simple timer and stopwatch electron app.

Release Download: [Mini Timer.zip](https://github.com/JamesHuangUC/Mini-Timer/raw/main/Release/Mini%20Timer.zip)

### Install

```
npm install
```

### Run

```
npm start
npm run dev (with nodemon)
```

### Package

```
npm run package-mac
npm run package-win
npm run package-linux
```
